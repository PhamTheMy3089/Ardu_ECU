# Đo DSO152 từng tầng mạch cảm biến RPM (KMZ10A) - 2026-07-19
#
# Đo trong quá trình bring-up mạch RPM. GND probe kẹp tại TP_GND (U15).
# Rotor quay chậm bằng tay/gió. Các giá trị dưới là số đọc thực từ DSO152.
#
# Diễn giải nhanh:
#  - A1 TP_5V: nguồn 5V ổn (4.94-5.12V), VREF ~2.5V -> nguồn analog OK.
#  - A2 TP_INA_OUT: sóng sin sau INA826 (Vpp 0.74-1.21V) -> cảm biến + INA OK.
#  - A3 TP_LMV358_OUT (U1): khi quay ra SÓNG VUÔNG 0.08-3.82V. Do tín hiệu mạnh
#    nên op-amp LMV358 bão hòa thành sóng vuông -> BÌNH THƯỜNG, không ép về sine.
#  - A5 TP_RPM_OUT (U2): sóng vuông sạch sau comparator LMV393.
#  - GPIO33 SAU KHI SỬA D1: khi quay lên tới ~3.36V sạch (log riêng). Bản đo dưới
#    còn thể hiện tình trạng CŨ khi D1 lỗi kẹp mức cao chỉ ~1.99-2.06V.
# ------------------------------------------------------------------------------

# filter=120us, edge=RISING, step=10us. Có cả đoạn coast-down (tắt starter).
#
# ==== KẾT LUẬN QUAN TRỌNG (chốt chẩn đoán) ====
#  1) RPM ghim ĐÚNG 1500 (raw=3/cửa sổ) suốt dải PWM 1150->1300us, KHÔNG đổi
#     dù mắt thấy động cơ quay nhanh hơn hẳn ở PWM cao. => Tần số xung bị khóa
#     cứng ~25Hz, KHÔNG bám theo tốc độ quay thật.
#  2) Nhưng khi TẮT starter (ESC OFF), rotor trôi chậm dần thì RPM GIẢM đều
#     thật: 1500 -> 750 -> 500 -> 375 -> 250 -> ... -> 0. Tức là khi ESC tắt,
#     cảm biến ĐỌC ĐÚNG tín hiệu magnet.
#  => Chỉ khi ESC/motor CHẠY thì reading mới bị khóa 1500; khi ESC tắt đọc đúng.
#     Đây là dấu hiệu kinh điển của NHIỄU EMI TỪ MOTOR/ESC lọt vào đường tín
#     hiệu RPM (~25Hz) lấn át tín hiệu magnet. Các số nhảy vọt 94k/106k/240k/
#     365k RPM là glitch xung đôi tức thời cùng nguồn nhiễu.
#  => KHÔNG phải lỗi firmware, KHÔNG phải bão hòa nam châm.
#
# HƯỚNG XỬ LÝ (ưu tiên):
#  1. Nối đất chung kiểu star ground (12V starter / UBEC 6V / mạch ECU).
#  2. Bọc shield + tách xa dây RPM khỏi dây motor/ESC; xoắn đôi RPM_OUT+GND.
#  3. Lắp lại D1 (TVS 3.3V đúng loại) + cân nhắc thêm tụ lọc nhỏ tại GPIO33.
#  4. Kiểm chứng: DSO tại U2/GPIO33 khi motor chạy -> nếu vẫn ~25Hz cứng = EMI.
#
# Ghi chú: khối ~200 dòng "NO_SIGNAL STAB=OFF" lúc chờ ở đầu đã được lược bớt
